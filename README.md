# Next.js App Boilerplate

Next.js app boilerplate.

## Installation

```
git clone --depth=1 https://github.com/k14lb3/next-app-boilerplate.git <project-name>
cd <project-name>
rm -rf .git
git init .
```

## Usage

Install the dependencies:

```
pnpm install
```

Run the development server:

```
pnpm dev
```

Build for production:

```
pnpm build
```
